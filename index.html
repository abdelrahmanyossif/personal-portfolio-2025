C - Simple Shell

The "0x16. C - Simple Shell" project in the ALX Software Engineering (SE) program is a capstone group project (me and Fady) that challenges students to build a simplified version of the UNIX shell. It's one of the most important and intensive projects in the C track, meant to consolidate many lower-level programming skills into a real-world tool.

GitHub Repository

C - Sorting algorithms & Big O

The "0x1B. C - Sorting algorithms & Big O" project in the ALX Software Engineering (SE) program is a foundational algorithmic project focused on implementing sorting algorithms in C and understanding their time and space complexity using Big O notation.

GitHub Repository

C - Binary trees

In this project, you will learn about binary trees, a fundamental data structure in computer science. You'll implement and manipulate binary trees in C, building your own tree structures, traversing them, and performing various operations such as insertion, deletion, depth/height calculations, and more. The goal is to understand how trees work from scratch—without using any external libraries or built-in tree functions.

GitHub Repository

AirBnB clone - The console

This is the first milestone project in the ALX Software Engineering (SE) program's AirBnB Clone series. In this project, you will build the command-line interface (CLI) for a simplified version of the AirBnB platform. This console is the core management system that allows users (and developers) to create, view, update, and delete objects related to the AirBnB clone, such as users, places, cities, etc. This project introduces key Object-Oriented Programming (OOP) concepts using Python, and it forms the backend foundation for future front-end and database integration.

GitHub Repository
